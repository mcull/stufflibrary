# Homepage item photos — drop zone

Drop one photo per item into this folder, named `<slug>.jpg` (`.png`/`.webp`
also fine). Any background works — the watercolor pipeline isolates the
object. Aim for ≥600px on the short side, one clear subject, no hands.

When the folder's loaded, tell Claude and the batch runs:
watercolor → blob (`homepage/watercolors/`) → homepage carousel.

Partial drops are fine — the batch skips missing slugs and can run again.
