# Lending Libraries of Things: A State-by-State Directory

> Draft page content for stufflibrary.org.
> Snapshot researched and verified July 2026.

## Framing (page intro)

Long before anyone put a lending library on the internet, people were running them out of church basements, firehouse garages, and public library branches. Tool libraries, toy libraries, gear libraries, "Libraries of Things" — real places, run mostly by volunteers and librarians, where your membership card is worth more than a garage full of stuff.

StuffLibrary exists to bring that same energy to the neighborhood scale, but we want to be plain about the order of operations: these places came first, and many have been at it for decades. Columbus, Ohio has run a tool library since 1976. Berkeley's has been lending since 1979. Los Angeles County has been lending toys to kids since 1935. If one of these is near you, join it — it's the fastest way to live the borrow-don't-buy life today, and most of them could use your membership.

This is a **snapshot, not a registry**: every entry below was checked against a live website in July 2026, but hours change, volunteer projects move, and new libraries open all the time. Know one we missed, or spot something stale? Tell us — this page gets better the way everything in the commons does: neighbor by neighbor.

**A note on what's here.** We favored places where an ordinary person can borrow a thing and bring it back: standalone tool libraries and Libraries of Things, plus the strongest public-library "Library of Things" collections in each state. We've also included some cousins worth knowing about — "tool banks" that lend only to nonprofits and community groups, statewide assistive-technology loan programs, and municipal cleanup-tool sheds — marked by type. Entries marked **UNVERIFIED** had signs of life we couldn't fully confirm; call before you drive.

**National starting points:**

- Local Tools / Tool Library Alliance map — the tool-library movement's own directory: https://localtools.org/find/ and https://www.toollibraryalliance.org/map
- myTurn-powered libraries (the software many run on): https://myturn.com
- USA Toy Library Association: https://usatla.org/toy-library-locations/
- Shareable's guides to starting your own: https://www.shareable.net

## Editorial notes (not for publication)

- Verification method: each "Verified: yes (July 2026)" entry had its site fetched live during research (week of July 6–10, 2026). UNVERIFIED entries are flagged inline.

- Public-library LoT collections are representative standouts, not exhaustive — many library systems have small unbranded collections.
