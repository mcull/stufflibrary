# Content Drop: Situating StuffLibrary in the Commons

This folder is a content-only contribution — five researched documents intended to become
public pages, plus one internal doc, plus a build spec. No application code is touched.

Researched and drafted July 2026. See `BUILD_SPEC.md` for how these become pages.

## Contents

**Public page drafts** (each file = framing intro + body + non-publishable editorial notes):

1. `directory-lending-libraries.md` — a July 2026 snapshot of real-world lending libraries
   of things across all 50 states + DC, verified against live websites. The "we're part of
   a real, physical movement" page.
2. `further-reading.md` — an annotated bibliography of ~35 landmark works (1960s–present,
   with a few older foundations) across commons, mutual aid, gift economy, consumption
   critique, social infrastructure, and sharing-economy threads. Genealogy + invitation.
3. `sharing-ecosystem.md` — a field guide to the online sharing world (Buy Nothing,
   Freecycle, repair cafés, timebanks, toy libraries...), with honest differentiation:
   they mostly gift, we lend.
4. `commons-not-tragedy.md` — the Ostrom page: plainspoken account of why the tragedy of
   the commons is optional, with her 8 principles mapped to platform features.
5. `faq.md` — FAQ, complete with Marc's answers (July 2026). One answer (liability)
   is flagged for attorney review before shipping — see its editorial notes.

**Internal:**

- `../internal/funding-sustainability.md` — research + recommendations for collective
  funding and structural guarantees of "non-commercial forever." A decision doc, not a page.
